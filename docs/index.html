Installation

Get up and running in seconds

Quick Install

curl -fsSL https://raw.githubusercontent.com/delapuente/abiogenesis/main/install.sh | bash

Installs Deno if needed, builds from source, and sets up PATH

Manual Install

git clone https://github.com/delapuente/abiogenesis.git
cd abiogenesis
cargo build --release
cp target/release/ergo ~/.local/bin/

Setup API Key

ergo --set-api-key sk-ant-your-key-here

Get your API key from Anthropic Console
